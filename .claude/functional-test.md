# 작동 테스트 가이드

## 목적
ContentSplitter의 모든 기능이 실제로 정상 작동하는지 하나씩 체크합니다.

## 준비 단계

### 1. 서버 시작
```bash
vercel dev
```

**예상 출력:**
```
Ready! Available at http://localhost:3000
```

### 2. 브라우저 열기
- URL: http://localhost:3000
- F12 → Console 탭 열어두기

---

## 기능별 테스트 체크리스트

### 1. 페이지 로드 테스트
**체크:**
- [ ] 페이지가 정상적으로 로드되는가?
- [ ] 헤더 "✨ ContentSplitter" 보이는가?
- [ ] 언어 선택 드롭다운 보이는가?
- [ ] 입력창 보이는가?
- [ ] 변환 버튼 보이는가?
- [ ] 콘솔에 빨간 에러 없는가?

**실패 시:**
- 콘솔 에러 메시지 복사
- 어느 파일/줄에서 에러 났는지 확인

---

### 2. 언어 변경 테스트
**체크:**
- [ ] 언어 드롭다운 클릭 → 4개 언어 보이는가? (한국어, English, 日本語, Español)
- [ ] English 선택 → UI 텍스트가 영어로 바뀌는가?
- [ ] 日本語 선택 → UI 텍스트가 일본어로 바뀌는가?
- [ ] Español 선택 → UI 텍스트가 스페인어로 바뀌는가?
- [ ] 한국어 선택 → 다시 한국어로 바뀌는가?

**확인 항목:**
- 헤더 부제목
- 플랫폼 이름 (인스타그램, 트위터 등)
- 버튼 텍스트 ("변환하기" ↔ "Convert")
- 글자 수 단위 ("자" ↔ "chars" ↔ "文字" ↔ "caracteres")

**실패 시:**
- 어느 언어에서 바뀌지 않는가?
- 어느 부분이 바뀌지 않는가? (헤더/버튼/레이블)

---

### 3. 입력 및 글자 수 카운터 테스트
**체크:**
- [ ] 입력창에 텍스트 입력 → 글자 수가 업데이트되는가?
- [ ] 한글 입력 → "N자" 표시되는가?
- [ ] 언어를 English로 변경 → "N chars" 표시되는가?
- [ ] 50자 미만 입력 후 변환 버튼 클릭 → 경고 메시지 나오는가?

**테스트 텍스트:**
```
짧은 텍스트
```
(50자 미만이므로 경고 나와야 함)

**실패 시:**
- 글자 수가 업데이트 안 되는가?
- 경고 메시지가 안 나오는가?

---

### 4. 플랫폼 선택 테스트
**체크:**
- [ ] 모든 체크박스가 기본으로 체크되어 있는가? (4개 모두)
- [ ] 체크박스 클릭 → 체크 해제되는가?
- [ ] 다시 클릭 → 체크되는가?
- [ ] 모든 체크박스 해제 후 변환 버튼 클릭 → 경고 메시지 나오는가?

**실패 시:**
- 어느 체크박스가 작동 안 하는가?
- 경고 메시지 내용이 맞는가?

---

### 5. 톤 선택 테스트
**체크:**
- [ ] 4개 라디오 버튼 보이는가? (전문적, 친근함, 캐주얼, 열정적)
- [ ] 기본값이 "전문적"으로 선택되어 있는가?
- [ ] 다른 톤 클릭 → 선택 바뀌는가?
- [ ] 한 번에 하나만 선택되는가? (라디오 버튼 동작)

**실패 시:**
- 여러 개가 동시에 선택되는가?
- 선택이 안 바뀌는가?

---

### 6. 해시태그 슬라이더 테스트
**체크:**
- [ ] 슬라이더가 보이는가?
- [ ] 기본값이 "10개"로 표시되는가?
- [ ] 슬라이더 움직임 → 숫자가 업데이트되는가?
- [ ] 최소값 0, 최대값 30인가?
- [ ] 언어 변경 시 단위가 바뀌는가? ("개" ↔ "" ↔ "個")

**실패 시:**
- 슬라이더가 움직이지 않는가?
- 숫자가 업데이트 안 되는가?

### 7. 예시 보기 버튼 테스트

### 8. AI 변환 테스트 (API 키 필요)

**⚠️ 주의:** 이 테스트는 실제 API를 호출하므로 비용이 발생합니다.

**체크:**
- [ ] 충분한 텍스트 입력 (50자 이상)
- [ ] 플랫폼 2-3개 선택
- [ ] 변환 버튼 클릭
- [ ] 로딩 섹션이 나타나는가?
- [ ] 로딩 메시지가 2초마다 바뀌는가?
- [ ] 30초 이내에 결과가 나타나는가?
- [ ] 선택한 플랫폼만 결과 카드가 생성되는가?
- [ ] 각 카드에 플랫폼 이름, 글자 수, 내용, 복사 버튼이 있는가?

**API 키 없이 테스트:**
- 변환 버튼 클릭 → 에러 메시지 나오는가?
- 에러가 사용자 친화적인가?

**실패 시:**
- 로딩이 계속되는가?
- 결과가 안 나오는가?
- 콘솔에 에러가 있는가?
